# MyProject – Java Servlet Web Application

## Description
This is a Java Servlet based web application.
It demonstrates servlet concepts, request–response handling, and basic CRUD operations.

## Technologies Used
- Java
- Servlets
- JSP
- JDBC
- Apache Tomcat
- HTML
- CSS

## Features
- Add student details
- View student records
- Update and delete records
- Simple web interface

## How to Run
1. Import the project into Eclipse
2. Configure Apache Tomcat Server
3. Run the project on server
4. Open browser and access the application
